# shellclip
Copy Reverse Shell Code Straight to Clipboard for Many Languages

## setup
	
	git clone https://github.com/montysecurity/shellclip.git
	cd shellclip/
	sudo apt install python3-pip
	pip3 install -r requirements

## languages supported
- Bash
- C
- PHP
- Python
- PHP + Python
- PHP + Python3
- PowerShell (PS1)
